Le Site dont vous êtes le Héros

Lecture des livres dont vous êtes le héros

Publié le 20 Décembre 2014

543

Dans un roulement de tonnerre, plusieurs stalactites, détachées par le choc, s'abattent autour de vous. Mais, heureusement, pour cette fois, aucune ne vous atteint. La secousse se propage toujours.

Voulez-vous courir droit devant vous, vers l'ouverture du tunnel qui semble maintenant toute proche

ou bien préférez-vous vous diriger lentement vers elle en regardant vers le haut pour éviter la chute éventuelle d'autres stalactites.
